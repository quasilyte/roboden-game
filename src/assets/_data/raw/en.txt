##menu.lobby.points_allocated : Points allocated

##menu.lobby.game_seed : Game seed
##menu.lobby.game_seed.description
The game seed controls the random world generation.
The same seed will create an identical world every time.
A seed should be a positive number.
If empty, a random seed will be used.

##menu.lobby.game_speed : Game speed
##menu.lobby.game_speed.description
The game speed multiplier.
x1.5 means the game will run 50% faster.
In other words, 10 minutes of the actual gameplay will progress the game time by 15 minutes.

##menu.lobby.fog_of_war : Fog of war
##menu.lobby.fog_of_war.description
This options toggles the fog of war.
If turned on, the map is not revealed from the start.

##menu.lobby.relicts : Relicts
##menu.lobby.relicts.description
Relicts are unique buildings that can be repaired by worker drones.
When repaired, they can serve the colony that captured them.

##menu.lobby.num_teleporters : Teleporters
##menu.lobby.num_teleporters.description
How many pairs of teleporters will be placed on the map.
If a colony lands on a teleportation platform, it will be transported to the connected teleporter.

##menu.lobby.players : Players
##menu.lobby.players.description
This option controls the number of allied colonies on the map and who controls them.
Selecting two players will enable a split-screen local multiplayer.
At least 1 gamepad is needed to play in the split-screen mode.
If any of the allied players is defeated, the game ends.
Only the single player mode score is ranked and can be published.

##menu.lobby.players.reverse : Players
##menu.lobby.players.reverse.description
In reverse game mode, it's possible to have a PvP experience when split-screen is enabled.
One player controls the dreadnought while another player controls the usual colony.
At least 1 gamepad is needed to play in the split-screen mode.
If any of the allied players is defeated, the game ends.
Only the single player mode score is ranked and can be published.

##menu.lobby.player_mode.single_player : single player
##menu.lobby.player_mode.single_bot : single bot
##menu.lobby.player_mode.player_and_bot : player and bot
##menu.lobby.player_mode.two_players : two players (split-screen)
##menu.lobby.player_mode.two_bots : two bots

##menu.lobby.ui_mode : User interface mode
##menu.lobby.ui_mode.description
Whether to show extra graphical user interface elements or not.
It's easier to play with them, but it's more immersive otherwise.

##menu.lobby.world_resources : World resources
##menu.lobby.world_resources.description
The number of resources the generated world has.
Fewer resources mean it will be hard to play a long battle.

##menu.lobby.gold_enabled : Gold
##menu.lobby.gold_enabled.description
Whether a map will have gold deposits.
On moon environment, disabling gold doubles the iron resources capacity.

##menu.lobby.world_size : World size
##menu.lobby.world_size.description
The generated world size.
It's usually harder to play on a smaller map.

##menu.lobby.oil_regen_rate : Oil regeneration rate
##menu.lobby.oil_regen_rate.description
How fast the oil (normal and red) resources regenerate over time.
This option can be used to completely disable that regeneration or make it faster.

##menu.lobby.num_creep_bases : Creep bases
##menu.lobby.num_creep_bases.description
The number of drone-producing bases the enemy has.
These bases produce more and more units over time, but they can't be replaced after being destroyed.

##menu.lobby.tech_progress_rate : Tech progress rate
##menu.lobby.tech_progress_rate.description
Tech progress rate specifies how much creeps tech level increase when they use a dedicated disk.
Faster tech progress rate makes the game easier for the dreadnought.

##menu.lobby.reverse_super_creep_rate : Super creeps rate
##menu.lobby.reverse_super_creep_rate.description
A chance of a unit released from a garrison being extra powerful.
The default rate is somewhat balanced.
A lower rate makes the game harder for the Dreadnought.

##menu.lobby.creep_spawn_rate : Creep spawn rate
##menu.lobby.creep_spawn_rate.description
This option controls the enemy unit spawn frequency.
It affects the late-game out-of-screen spawns only.
The enemy factories will produce units at the same speed.

##menu.lobby.drones_power : Colony drones power
##menu.lobby.drones_power.description
The colony drones health multiplier.
Higher values make the game for the creeps harder.

##menu.lobby.creeps_difficulty : Creeps power
##menu.lobby.creeps_difficulty.description
The creep units' power multiplier.
It mostly affects the amount of durability these units have.

##menu.lobby.arena_progression : Wave difficulty progression
##menu.lobby.arena_progression.description
This option defines the difficulty progression speed.
Every wave is harder than the previous one and this option controls by how much.

##menu.lobby.initial_creeps : Initial creeps count
##menu.lobby.initial_creeps.description
How many creep units will be placed on the map from the very beginning.
This option affects how easy the early game exploration for the colonies will be.

##menu.lobby.creep_fortress : Creep fortress
##menu.lobby.creep_fortress.description
The fortress makes it hard to explore the region it's located in.
It produces unique sentinel drones to patrol the area.
It does not attack the colonies if they're out of its range.

##menu.lobby.ion_mortars : Ion mortars
##menu.lobby.ion_mortars.description
Whether creeps have ion mortar pits available.
The ion mortar pits have a very long range, but low accuracy.
Their attacks deal medium damage and burn a lot of the drone's energy.
These mortars can't attack ground targets.

##menu.lobby.boss_difficulty : Dreadnought power
##menu.lobby.boss_difficulty.description
The dreadnought power level.
It affects the number of units it will produce and how hard it will be to defeat it in battle.

##menu.lobby.super_creeps : Super enemies
##menu.lobby.super_creeps.description
Whether enemies can have super versions of units.
This is similar to the elite drones the player may have.
Turning it on makes the game significantly harder for the colonies.

##menu.lobby.starting_resources : Starting resources
##menu.lobby.starting_resources.description
This defines whether starting colony will have any starting resources.
Some drone builds can benefit from this early game advantage.

##menu.lobby.land : Terrain
##menu.lobby.land.description
Choose how many mountains and landcracks the map will have.
Make them really rare or appear almost everywhere.

##menu.lobby.environment : Environment
##menu.lobby.environment.description
Switches between biome types.

Apart from the visual differences, each biome has
some special differences that will affect the gameplay.

##menu.overview.intro_mission
Intro mission (est. time: 15 minutes)

This is a guided quick-start mode. Learn the gameplay fundamentals and defeat the opposing force.

Complete this mission to unlock other game modes.

##menu.overview.classic
Classic mode  (est. time: 35 minutes)

Win by defeating the enemy flagship, the dreadnought.

The dreadnought is very powerful. Evade the combat with it until allied colonies are evolved enough.

Split-screen multiplayer: cooperative.

##menu.overview.arena
Arena mode (est. time: 60 minutes)

Win by surviving for 20 waves.

Split-screen multiplayer: cooperative.

##menu.overview.inf_arena
Infinite arena mode (est. time: 90 minutes)

Stand the ground for as long as possible.

It's not possible to win in this mode, but the score is based on the playtime.

Split-screen multiplayer: cooperative.

##menu.overview.reverse
Reverse mode (est. time: 40 minutes)

Play as the other side and lead the dreadnought.

Win by destroying all colonies.

Split-screen multiplayer: competitive (PvP).
